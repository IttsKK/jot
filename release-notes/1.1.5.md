## Highlights

- Added a Check for Updates action in Settings so installed builds can trigger Sparkle update checks without going through the menu bar.
- Cleaned up the Settings UI with more consistent rows and controls, and fixed the default queue picker so Notes can be selected again.
- Made the main app window behave independently from the menu bar workflow so closing or quitting the app UI leaves quick capture, the menu bar item, and the Today list available in the background.

## Improvements

- Hardened the menu bar icon with template rendering and visible fallbacks for machines where the primary symbol does not appear correctly.
- Clarified full-process termination in the menu bar as `Quit Jot Completely` so it is distinct from closing the main app UI.
